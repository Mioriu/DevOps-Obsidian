**Vim** — это расширенная версия классического редактора `vi`, широко используемая в Unix-системах. Он известен своей эффективностью, мощью и возможностью тонкой настройки. В этом уроке мы познакомимся с базовыми концепциями и режимами работы Vim, которые станут фундаментом для последующих уроков по редактированию, навигации, визуальному режиму и настройке редактора.

### Что такое Vim и его преимущества

Vim представляет собой текстовый редактор, работающий полностью в терминале (хотя существует и графическая версия — gVim). Ключевые особенности Vim:

- Многообразие режимов работы (нормальный, вставки, визуальный, командный) для оптимизации процесса редактирования;
- Высокая скорость и эффективность при работе с текстом;
- Гибкость в настройке и возможность расширения за счёт плагинов;
- Поддержка скриптов и автоматизация рутинных задач.

### Режимы работы в Vim

Главной особенностью Vim является его режимность. Основные режимы:

- **Нормальный режим (Normal mode)**: режим по умолчанию. Здесь вы перемещаетесь по тексту, удаляете или копируете строки, выполняете команды редактирования. От этого режима можно перейти в другие режимы.
- **Режим вставки (Insert mode)**: используется для ввода и редактирования текста. В него можно войти из нормального режима, нажав клавишу `i` (вставка перед курсором) или `a` (вставка после курсора). **Чаще всего для входа в режим вставки используется клавиша `i`.**
- **Визуальный режим (Visual mode)**: позволяет выделять текст для копирования, вырезания или изменения его оформления. Подробнее о визуальном режиме мы рассмотрим в следующих уроках.
- **Командный режим (Command mode)**: здесь выполняются команды по сохранению, выходу, поиску, замене текста и другие действия. Для входа в этот режим нажмите `:` из нормального режима.

### Запуск и выход из Vim

Чтобы запустить Vim, откройте терминал и выполните:

```nginx
vim filename.txt
```

Если файла с именем `filename.txt` не существует, Vim создаст его при сохранении. После открытия файла в Vim:

- Нажмите `i` для входа в режим вставки и начала редактирования.
- Нажмите `Esc` для возврата в нормальный режим.
- Чтобы сохранить изменения, в нормальном режиме наберите `:w` и нажмите `Enter`.
- Чтобы выйти из Vim, наберите `:q` и нажмите `Enter`. Если вы хотите сохранить и выйти одновременно, используйте `:wq` или `:x`.
- Если изменения не нужны, и вы хотите выйти без сохранения, наберите `:q!`.

> **Команда `:line-number` (например `:25`) позволяет перейти к указанной строке.**

### Базовые команды навигации

Навигация в Vim происходит в основном в нормальном режиме. Вот несколько базовых команд:

- **h, j, k, l** — перемещение курсора влево, вниз, вверх и вправо соответственно.
- **0** — переход к началу строки.
- **$** — **переход к концу строки.**
- **gg** — переход в начало файла.
- **G** — переход в конец файла.

Эти команды помогут вам быстро перемещаться по тексту. В последующих уроках мы углубимся в возможности навигации, включая продвинутые команды.

### Краткое резюме

- Vim — мощный текстовый редактор, требующий освоения нескольких режимов для эффективного редактирования.
- Ключевые режимы: нормальный, вставки, визуальный и командный.
- Основные команды запуска, сохранения и выхода: `vim filename.txt`, `i` для вставки, `:w` для сохранения, `:q` для выхода
