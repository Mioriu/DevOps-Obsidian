

Визуальный режим в Vim позволяет выделять блоки текста для их последующего редактирования, копирования, перемещения или удаления. Он делает работу с текстом более гибкой и точной. В этом уроке мы рассмотрим, как войти в визуальный режим, какие виды выделения доступны, и как использовать команды для манипулирования выделенным текстом.

### Вход в визуальный режим

Для активации визуального режима в Vim существует несколько вариантов, каждый из которых предназначен для разных случаев выделения текста:

- **v** — переключение в посимвольный визуальный режим. Этот режим позволяет выделять текст по одному символу.
- **V** — строковый визуальный режим. Он выделяет целые строки, что удобно для операций над строками целиком.
- **CTRL + v** — блочный (колоночный) визуальный режим. С его помощью можно выделять прямоугольные области текста, что полезно для редактирования колонок или табличных данных.

### Работа в визуальном режиме

После входа в визуальный режим выделенный текст можно редактировать различными способами:

- **y** — копирует (yank) выделенный текст в буфер обмена.
- **d** — удаляет выделенный текст, сохраняя его в буфере (как при вырезании). **Это аналог команды «вырезать» в других редакторах: текст исчезает с места и остаётся доступным для вставки через p или P`.**
- **c** — удаляет выделенный текст и переводит редактор в режим вставки для ввода нового текста.
- **~** — переключает регистр выделенного текста, меняя прописные буквы на строчные и наоборот.
- Другие команды, такие как `>` и `<`, позволяют изменять отступы выделенного блока текста.

### Выход из визуального режима

Чтобы выйти из визуального режима и вернуться в нормальный режим, просто нажмите клавишу `Esc`.

### Примеры практического использования

- **Копирование целой строки:**
    
    ```1c
    V       " Вход в строковый визуальный режим
    y       " Копирование выделенной строки
    ```
    
- **Удаление посимвольного фрагмента:**
    
    ```1c
    v       " Вход в посимвольный визуальный режим
    [переместите курсор, выделяя нужный фрагмент]
    d       " Удаление выделенного фрагмента
    ```
    
- **Блочное выделение для копирования колонок текста:**
    
    ```1c
    CTRL+v  " Вход в блочный визуальный режим
    [используйте стрелки для выделения блока]
    y       " Копирование выделенного блока
    ```
    

### Практические советы

- Применяйте визуальный режим для точного выделения областей текста, когда требуется работать не с целыми строками, а с отдельными частями строк или колонками.
- Экспериментируйте с разными типами визуального режима, чтобы понять, какой из них лучше всего подходит для конкретных задач.
- Комбинируйте визуальный режим с командами копирования, удаления или изменения, чтобы ускорить процесс редактирования и минимизировать ошибки.
