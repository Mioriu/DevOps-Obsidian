Любой Linux-дистрибутив состоит из сотен и даже тысяч различных программ и библиотек. Управлять всем этим вручную было бы очень неудобно. Именно поэтому существуют **пакетные менеджеры**, которые помогают устанавливать, удалять и обновлять программное обеспечение простыми командами.

### Что такое пакетный менеджер?

**Пакетный менеджер** — это специальная программа, которая отвечает за установку, удаление, обновление и управление программами (пакетами) в Linux. Основные функции пакетных менеджеров:

- Поиск программного обеспечения в репозиториях
- Установка и удаление ПО
- Разрешение зависимостей (установка дополнительных библиотек)
- Обновление установленного ПО и системы

### Какие бывают пакетные менеджеры?

Наиболее популярные пакетные менеджеры зависят от используемого дистрибутива Linux:

- **APT (Advanced Package Tool)** — Debian, Ubuntu и их производные
- **YUM/DNF** — CentOS, Fedora, RHEL
- **Pacman** — Arch Linux
- **Zypper** — openSUSE

Рассмотрим подробнее работу самых распространённых менеджеров: **APT** и **YUM/DNF**.

### ![](https://ucarecdn.stepik.net/fe3ae3ee-b932-43e9-9632-b5d7f68ad656/)

**APT** используется в дистрибутивах Debian и Ubuntu. Примеры базовых команд:

- Обновить список пакетов из репозиториев:
    
    ```bash
    sudo apt update
    ```
    
- Обновить установленные пакеты до последней версии:
    
    ```bash
    sudo apt upgrade
    ```
    
- Установить новый пакет (например, vim):
    
    ```bash
    sudo apt install vim
    ```
    
- Удалить установленный пакет:
    
    ```bash
    sudo apt remove vim
    ```
    
- Удалить пакет с его конфигурацией:
    
    ```bash
    sudo apt purge vim
    ```
    
- Искать пакет по имени или описанию:
    
    ```sql
    apt search пакет
    ```
    
- Вывести подробную информацию о пакете:
    
    ```sql
    apt show vim
    ```
    

### ![](https://ucarecdn.stepik.net/b9c174ef-a048-4430-a02b-e3b2c9a1e2ec/)

В CentOS, Fedora и Red Hat используются менеджеры **YUM** или более современный **DNF**. Вот их основные команды:

- Обновить список пакетов:
    
    ```sql
    sudo dnf check-update
    ```
    
- Обновить установленные пакеты:
    
    ```bash
    sudo dnf upgrade
    ```
    
- Установить новый пакет:
    
    ```bash
    sudo dnf install vim
    ```
    
- Удалить пакет:
    
    ```bash
    sudo dnf remove vim
    ```
    
- Поиск пакета в репозиториях:
    
    ```sql
    dnf search пакет
    ```
    
- Информация о пакете:
    
    ```nginx
    dnf info vim
    ```
    

Для YUM используются аналогичные команды, достаточно заменить `dnf` на `yum`.

### ![](https://ucarecdn.stepik.net/05e5f5c6-a473-4af2-b560-9d37cf5f2bce/)

Одна из главных задач пакетного менеджера — это автоматическое разрешение зависимостей. Например, если вы хотите установить пакет, зависящий от других библиотек, пакетный менеджер автоматически установит все необходимые компоненты:

```bash
sudo apt install git
```

APT самостоятельно найдёт и установит все библиотеки, нужные для работы Git.

### ![](https://ucarecdn.stepik.net/9fbcde28-18d6-44e7-9d64-6d22e5001a3b/)

Пакетные менеджеры могут удалять автоматически установленные пакеты, которые больше не нужны системе:

```bash
sudo apt autoremove    # удаляет пакеты-зависимости, которые больше не нужны
sudo apt clean         # очищает кэш .deb файлов
```

- Для APT:
    
    ```bash
    sudo apt autoremove
    ```
    
- Для DNF/YUM:
- ```bash
    sudo dnf autoremove
    ```
    

### ![](https://ucarecdn.stepik.net/7dc01656-6f7a-4c63-9f9e-b3cdfe891c61/)

Полезные команды для управления системой:

- Показать список всех установленных пакетов:
    
    ```applescript
    # Debian/Ubuntu
    apt list --installed
    
    # Fedora/CentOS
    dnf list installed
    ```
    
- Очистка кеша загруженных пакетов:
    
    ```bash
    # Debian/Ubuntu
    sudo apt clean
    
    # Fedora/CentOS
    sudo dnf clean all
    ```
