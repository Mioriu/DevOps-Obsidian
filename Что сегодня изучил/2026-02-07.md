Сегодня изучал утилиту awk
awk - мощная утилита для работы с текстом, позволяет выводить только определенные строки, поля строк, номера, и т.д. Позволяет сравнивать отдельные поля строк, можно использовать циклы, функции для более гибкой работы с текстом.
Шаблон работы:
awk 'условие {действие}' файл

Примеры:
awk '{print $1}' users.txt -  Выведет только первое поле из всех строк, по умолчанию разделитель пробел
awk -F',' '{print $2}' data.csv - выведет второе поле из всех строк, используя разделитель запятую
awk '{printf "%-10s %-10s %3d\n", $1, $2, $3}' users.txt - можно использовать форматирование как в C/Java
В целом мощный инструмент для работы с текстом, даже можно использовать циклы, считать различные параметры, на вход принимает любой текст, текстовые файлы, выводы других команд и т.д.
Утилита sort, часто используется вместе с uniq, нужна для сортировки текста, так же можно сортировать числа, но если не указать флаг sort -n, то будет сортировать числа по алфавиту: 1,10,2,20 и т.д. uniq позволяет фильтровать дубликаты, например uniq -d выводит только повторяющиеся, uniq -u выводит только уникальные.
Все эти команды нужны для работы со строками и их можно комбинировать через пайплайн - |
Проходил утилиты для сравнения файлов:
diff - построчное сравнение
comm - сравнение отсортированных файлов построчно
comp - быстрое побайтовое сравнение
Примеры использования:
diff file1 file2
у diff есть различные флаги, в целом любую утилиту можно посмотреть через --help
Начал изучать bash, пока из интересного, комментарий в самом начале любого баш скрипта:
#!/bin/bash - показывает каким шелом/интерпретатором будет выполняться скрипт.
Переменные $0,$1,$2 хранят в себе аргументы скрипта, например:
./script.sh 1 2 3 - $0 - название скрипта, $1 - первый аргумент и так далее
