Сегодня прошел не много, а именно историю команд в линукс и команду grep с регулярными выражениями.
В линукс мы можем вывести всю историю последних команд:
history
history -c - удалить всю историю команд
history -w - удалить историю команд в текущей сессии 
history -d - удалить из истории определенную команду
История команд в bash хранится в файле .bash_hisotry, в других шелах например zsh в .zsh_history.
Есть переменные окружения, которые содержат некоторые параметры истории, например:
HISTSIZE - содержит значение, которое определяет размер истории команд хранимых в памяти
HISTFILESIZE - содержит значение, которое определяет количество команд хранящееся в файле .bash_history
Команды можно переключать стрелками, стрелка вверх - предыдущая команда, вниз - в обратную сторону 
Еще немного о history, после прописания команды мы увидим список последних команд, тут можно нажать SHIFT+R и начать вводить команду, после чего нам будет предложена подходящая команда. Так же при выводе history показаны номера команд, их можно выводить через !123 (!№команды)

GREP
Команда grep позволяет осуществлять построчный поиск по файлу или например выводу другой команды, часто используется через конвейер | , например 
cat /var/log/syslog | grep error
Можно вызывать для файлов grep "error" log.txt, для каталогов grep -r "test" /var ,
можно выводить названия самих файлов при поиске по множеству файлов grep -l - отбражает только имена, grep -H -отображает имена файлов перед найденной строкой, -n - указывает номера строк
В grep можно использовать регулярные выражения что дает очень большие возможности для поиска строк по любым паттернам/шаблонам.
Различных паттернов поиска в регулярных выражениях очень много, не вижу смысла запоминать, это все нужно использовать на практике, думаю если что можно их генерить через ИИ.
